para la prediccion de los pasos visibles de los satelites se puede usar bajo linux gpredict o un soft de windoes que se llama Wxtrack y corre bajo wine en mac o linux.

http://gpredict.oz9aec.net/screenshots.php
http://www.satsignal.eu/software/wxtrack.htm

hay un software nuevo que funciona en cualquier plataforma via web es:
http://mada.la.coocan.jp/sat/index.htm?lang=en

igualmente via web y para sacar tablas de predeccion de los pasos visibles:
http://www.n2yo.com/

o, cuando hay suerte y funciona:
http://spaceweather.com/flybys/

si se trata solo de saber donde esta la ISS:
http://www.lizard-tail.com/isana/tracking/index.html
