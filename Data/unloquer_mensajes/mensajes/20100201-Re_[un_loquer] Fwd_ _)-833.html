>

>
> Puedes también presionar el botón izquierdo del mouse.
>
> Borrar fotogramas
>
> Presione la barra espaciadora
>
> También puede utilizar el botón derecho del mouse.
>
> Grabar en archivo de video
>
>  Presione la tecla S para guardar el clip actual en su disco duro.
>
> Se graba en el directorio: Documents/toonloop/new_project
>
>  Resumen de control en el teclado.
>
> Toonloop es un software interactivo. Mientras corre puede controlarse usando
> botones del teclado. Aquí se encuentra una lista de controles con las
> respectivas funciones para controlar Toonloop.
>
> 1.      Barra espaciadora: grabar un fotograma
>
> 2.      Suprimir o barra espaciadora para borrar el ultimo fotograma.
>
> 3.      “R” para remover todos los fotogramas del clip.
>
> 4.      “S” para grabar la secuencia actual como Motion-JPEG movie (formato)
>
> 5.      “I” para mostrar el numero del fotograma, el numero de fotogramas en
> el clip y la velocidad de los mismos.
>
> 6.      “H” para mostrar el mensaje de ayuda.
>
> 7.      “Arriba” (flechas) para aumentar la velocidad de los fotogramas del
> clip.
>
> 8.      “Abajo” (flechas) para disminuir la velocidad de los fotogramas del
> clip.
>
> 9.      “A” para encender o apagar las grabaciones automáticas. (Graba un
> fotograma en cada fotograma, es un intervalometro). Mejor para crear clips
> con lapsos tiempo automáticos)
>
> 10.    “K” o “J” para aumentar o disminuir la velocidad de grabado
> automático.
>
> 11.    “F”  o Esc para pasar a modo de pantalla completa.
>
> 12.    “Shift-Q” para salir.
>
> 13.    “.” Para cambiar el tema gráfico.
>
> 14.    “TAB” para cambiar la dirección de reproducción.
>
> 15.    Presione un numero entre el 0 y el 9 para cambiar a otro clip de
> película.
>
> 16.    “P” para pausar la reproducción.
>
> 17.    “O” para activar o desactivar el papel de cebolla.
>
> 18.    “B” para tomar una instantánea como nueva imagen de fondo.
>
> 19.    “N” para seleccionar el próximo efecto disponible.
>
> 20.    “izquierda” (flechas) para mover el cursor a la izquierda.
>
> 21.    “Derecha” (flechas) para mover el cursor a la derecha.
>
> 22.    Presione “retorno” para mover el cursor al último fotograma. (al
> final)
> - Press Return to move the write head to the last frame. (at the end)

>> Add frames to a clip
>>
>> Press the space bar to add a frame. You can also press on the left
>> button of the mouse.
>> Delete frames
>>
>> Press on the Backspace key. You can also use the right mouse button.
>> Save to a movie file
>>
>> Press on the S key to save the current clip to you hard drive. It is
>> saved in the Documents/toonloop/new_project folder in your home
>> directory.
>> Keyboard Controls Summary
>>
>> Toonloop is an interactive software. While it runs, or can control it
>> using keyboard keys. Here is a list of keystrokes to control Toonloop.
>>
>>
>> Toonloop interactive keyboard controls :
>> - Press the SPACE bar to grab a frame.
>> - Press DELETE or BACKSPACE to delete the last frame.
>> - Press 'r' to reset and start the current sequence.
>> (and remove all its frames)
>> - Press 's' to save the current sequence as a Motion-JPEG movie.
>> - Press 'i' to print current loop frame number, number of frames in loop
>> and global framerate.
>> - Press 'h' to print a help message.
>> - Press UP to increase frame rate.
>> - Press DOWN to decrease frame rate.
>> - Press 'a' to toggle on/off the auto recording.
>> (it records one frame on every frame) It is an intervalometer.
>> Best used to create timelapse clips automatically.
>> - Press 'k' or 'j' to increase or decrease the auto recording rate.
>> - Press 'f' or ESCAPE to toggle fullscreen mode.
>> - Press SHIFT-'q' to quit.
>> - Press '.' to change the graphical theme.
>> - Press TAB to change the playback direction.
>> - Press a number from 0 to 9 to switch to a different clip number.
>> - Press 'p' to pause playback.
>> - Press 'o' to toggle the onion skinning on/off.
>> - Press 'b' to take a snapshot as the new background image.
>> - Press 'n' to select the next effect available.
>> - Pres the Left arrow to move the write head to the left. (towards the
>> beginning)
>> - Press the Right arrow to move the write head to the right. (towards the
>> end)
>> - Press Return to move the write head to the last frame. (at the end)
>>
