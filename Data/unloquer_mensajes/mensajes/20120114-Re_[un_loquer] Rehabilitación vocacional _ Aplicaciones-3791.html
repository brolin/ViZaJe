Tratemos de abstraer un taller, sin nombres propios .. entonces tenemos varios genéricos que van a ir VARIANDO ej: $personas, $sitios, $talleres, $moods, etc. y que escribiremos con el prefijo $ para facilidad en la futura escritura del pseudocódigo.

Estas VARIABLES van a relacionarse entre ellas ejerciendo FUNCIONES, ej: mandarInfo(), buscarSitio(), registroFotográfico(), etc. estas Funciones tienen unos atributos, Ej: la Info no se manda sola entonces quiza hay que decir quien va a mandar la info y como hay varios tipos de info, entonces especificar el tipo de info.
Este caso se podría ver asi:

var $persona = "fede";
var $mood = "animoso, con tiempo libre";
var $taller = "Escucha de satelites";
var $info = "presupuesto";

Function mandarInfo($persona, $info) {

}
