hola todos,

por razones de nettiquete (horrible palabra) y de economia, esta lista no deja pasar adjuntos de mas de 100k, de esta manera la convertimos en un buzon de imagenes de aca y de alla. Una lista de correos es primordialmente para texto, normalmente para discutir, compartir, informar. Todos ese tipo de anuncios en HTML que incluyen imagenes no son mas que una pesima practica algunas veces clasificada como spam.

un comunicado bien puede incluir imagenes, si estas se comprimen apropiadamente 100k seran mas que suficiente.

en el caso de el logo o de un pdf, pues sera mejor subir dichos archivos a alguna pagina web... desde flickr a imageshack o http://www.rapidshare.com/ pueden servir cuando no se tiene una cuenta de ftp. o mandar dicho archivo a las personas que se considera deben recibirlo.

la lista de unloquer ha sido suscrita, sin nosotros pedirlo, a la lista de correos de:

Bienvenido a la lista de distribucion Traxmision@indivia.net Para mandar un mensaje a esta lista, env=EDelo a: traxmision@indivia.net

esto es tambien una mala practica. y no lo es poque no valoremos la informacion que publican, al contrario, es vital, ahora hacen una conexion con oaxaca :) pero no podemos tener una lista que queremos sea para discutir y crecer el proyecto de unloquer convertida en un tablon de anuncios de otras listas que ya recibimos.
