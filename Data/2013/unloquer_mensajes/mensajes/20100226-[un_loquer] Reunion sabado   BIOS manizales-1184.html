Informacion tecnica detallada sobre el proyecto:

gstreamer:
En Linux no existía una API (interfaz para programacion de aplicaciones ) completa para soporte de Multimedia, al formarse el proyecto Freedesktop, se aunaron los esfuerzos para producir un sistema (API) orientado completamente para Multimedia,  El resultado, desde 1999, es GStreamer un paquete completo para la creación de aplicaciones orientadas a Multimedia, como players de MP3, video, etc.  

Haciendo uso de sus gran versatilidad y flexibilidad, se utilizo GStreamer para la creación de un video wall o mega pantalla a partir de la union de varias pantallas, la ventaja que tiene sobre productos que venden en el mercado para este fin es su infrastructura de bajo coste, basado principalmente en una red de distribucion de datos ethernet, y la reutilizacion de equipos ya en desuso por sus limitaciones.

Asi, reuniondo las pantallas como una solo y con una programacion basica disponible en la web se hace el despliegue de video la pantalla -  multipantalla, en su nombre se incluyen Video Wall y video Whale.

cron: 
Es un programador de tareas que nos sirve para sincronizar la ejecucion de scripts o archivos de audio con el reloj interno del computador, para hacer alarmas, lecturas programadas, o reporducir piezas de audio multipistas en diferentes maquinas.

ssh: 
Es la forma de conexion remota mas difundida en los ambientes Linux/Unix, este nos permite tener control total o parcial de una maquina remota, dependiendo de los privilegios que posea el usuario con el cual nos conectamos. Las comunicaciones van cifradas en este protocolo garantizando la seguridad. Para el caso de BIOS, es una herramienta fundamental pues podemos instalar o configurar aplicaciones o servicios sin tenernos que desplazar fisicamente maquina por maquina, tambien nos permite transferir ficheros de una maquina a otra agilizando la puesta en marcha de la instalacion. 

sooperlooper: 
Programa que sobreponer capas de audio a una base grabada por el usuario. Generando bucles o loops en tiempo real que sirven para acompañar las animaciones generadas con ToonLoop. 

toonloop:
Este es un sencillo programa de animacion stopmotion (cuadro a cuadro) que con webcams tipo V4L2 nos permite generar clips .mov de las secuencias de imagenes que vayamos capturando. Tiene la opcion de piel de cebolla para el ultimo cuadro capturado que nos da mayor precision a la hora de hacer una animacion mas elaborada ademas muchas otras funciones para publicos mas avanzados, pero para un publico principiante no deja de ser de facil uso.

puredata:
es un software que nos permite escribir nuestros propios programas basados en el paradigma del flujo de datos (dataflow), es una interface muy similar a la de los flujos electronicos donde uno conecta un modulo con otro, cada modulo desarrolla una tarea especifica y al final se tiene un programa. PureData puede ser utlizado para la manipulacion de sonido y video en tiempo real. en BIOS lo usamos entre otras para interfacear SoonperLooper y ToonLoop en un patch que llamamos SooperTooner. Igualmente hay un par de parches (o programas) mas para generar bits de video.

puredyne:
Basados en un LiveCD (Sistema Operativo que funciona desde un CD-ROM) desarrollamos una version llamada planktum-BIOS que nos permite tener todo el software arriba descrito en un solo sistema facil de instalar y transportar. puredyne es entonces la manera como accedemos a los repositorios de programas de Ubuntu para generar nuestra propia distribucion de software libre.
