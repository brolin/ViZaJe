al momento, por los streams :
data = OSC messages (control, text)
audio,video = icecast2 streams

• puedes encontrar algunos how-to guides en :
http://wiki.theartcollider.org/inout/doc
• ex. para hacer screen capture (sent on the network) on os x:
http://wiki.theartcollider.org/tools/oggzcap
• ex. para enviar live mic/webcam from browser: http://theartcollider.org/flex/
• command line tools: http://wiki.theartcollider.org/inout/usage

no sé si es lo mismo ahora, pero hace 2 años, habia un "vocabulario"
OSC para eligir/cambiar de stream, un poco como:
/osc/inout/me/hola-yo-soy 87.12.12.34
/osc/inout/me/tengo-video 87.12.12.34/icecast-video
/osc/inout/me/tengo-audio 87.12.12.34/icecast-audio
    (arriving on network)
/osc/inout/other/here-is-video 132.12.17.34/icecast2stream
    (you can accept the stream from a randomly choosen other)
/osc/inout/me/quiero-otro
    (or ask for another...)
